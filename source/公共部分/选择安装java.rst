为啥需要安装java
==================================================

mc是java开发的， 相关的一些开服程序都是需要java环境才能运行的。 

我要选择哪个版本
==================================================
java的版本是比较多的， 具体的版本选择还是需要看我们部署的服务端或者客户端的需求来。 一般以具体的官网的版本为主。
